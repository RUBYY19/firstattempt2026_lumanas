Project Description

This project is a web-based application developed using HTML, CSS, and Vanilla JavaScript.
The purpose of this project is to demonstrate basic web development concepts such as page structure, styling, and interactive features using pure JavaScript without relying on external frameworks.

The application was built based on the design submitted in Activity #10, converted into a fully functional web interface.

Installation
Clone the Repository

Using GitHub, clone the repository to your computer.

git clone https://github.com/<your-username>/firstattempt2026_lumanas.git

Navigate to the project folder.

cd firstattempt2026_lumanas
Run the Project

Since the project uses Vanilla JavaScript, no additional installations or frameworks are required.

Steps to run the project:

Open the project folder.
Locate the file named:
index.html
Double-click the file or open it using any web browser such as Google Chrome or Microsoft Edge.

The application will load and run directly in the browser.
